  3. 切换账号bug

  4. 一个修改好包名为白名单内包名，通过AndroidStudio等工具编译的少年派破解专用app

  5. SNP服务器地址

二、破解

  1. 利用切换账号bug登入账号。

  2. 进入书包找到一个。

  3. 根据网上教程配置好wifi热点和fiddler。

  4. 将少年派破解专用app上传至服务器。
