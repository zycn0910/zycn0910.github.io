如何利用python实现微信自动回复？

2022-03-12
小教程-干货

前言

流水、飞鸟、旅途，唐人用汉字计算着未知，计算着千年万里，计算着一年一月，却计算不出你什么时候才能回复我的信息。

听说你还在为打游戏时没有秒回 男 女朋友 的信息而些许神伤？

秒回信息！从现在开始！

step1

首先我们需要一个小工具python和python依赖包itchat，因为python与其依赖包的安装方法已经有很多人发过了，所以就不再重复造轮子了。

step2

在任意位置新建一个.py文件，导入itchat依赖包import itchat后使用itchat文档里的注册函数，注册微信监听进程。

@itchat.msg_register('Text') # Text为文本信息参数，此参数不会回复表情、图片等信息

进程成功注册之后，python将持续监听已登录微信收到的信息，但此时是不能查看信息的。如果想要查看接受到的信息需要把信息提取出来，但是由于python为命令行形式，所以只能查看文本信息或者emjo表情代码。

Message = msg['Text'] # 取出微信收到的信息的文字内容

通过reply可以设置每次好友发送信息后回复的内容。

replay = u'需要回复的内容' # u为python中的转义符号，表示单引号内的文本、符号仅取字面意，无其他含义。

然后通过内置的itchat.auto_login()函数登录账号即可，python会在后台自动监听微信收到的信息。

每次回复的内容都一样？ 男 女朋友肯定会怀疑回复的真实性

所以我们可以使用if and elif来判断微信收到的信息，进行指定内容回复。

if u'你在干什么' in Message:
    replay = u'在想你呢'
elif u'？' in Message: # 因为女生发的标点符号可能会有言外之意，所以建议单独设置
    reply = u'怎么了'
eilf u'关键词' in Message:
    reply = '回复内容'

综上所述，就是实现微信自动回复的全部步骤了，由于python的灵活性和微信可以给自己发送信息的特性，所以此项目可实行需要远程控制的部分项目，例如：自动超星直播课（只需把直播邀请码或者直播间地址发送给自己，即可好好学习，天天向上）
